Inexplicably, this process takes days, so the IT organization has provided me with a loaner. They were kind enough to pre-load Xcode. Yesterday morning when I sat down to do some coding, I typed the following:
$ ionic start myapp-bash: ionic: command not found

So now I find myself starting from scratch with my development environment. And I'm sure it will happen sometime this week (or next) when I swap out the loaner and get mine back. To avoid the pain of figuring this out all over again, here are all of the things I need to install,  how to install them, and in what order, to make myself a more effective developer.



$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"$ brew doctor$ brew install node$ sudo npm install -g azure-cli$ sudo npm install -g grunt grunt-cli$ sudo npm install -g cordova ionic
